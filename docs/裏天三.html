裏天三
（うらあまさん）
所在地茨城県つくば市天久保三丁目の北西部
Category 筑波大学の俗語
Class 地域名

概要

裏天三とは天久保三丁目北西部のこと。要の一部も含む。 天久保三丁目は筑波大学の敷地が大半を占めるが、北東部と北西部のみは通常の住宅や店舗が並んでいる。 このうち、北東部は店舗が多く天三の『表』に当たると考えられ略称天三はここに限定することも多い。一方、相対的に店舗が少ない北西部が裏天三と呼ばれる。 番地は2～7にあたる。 また、「あじよし」など隣接する要の一部も裏天三の一部と考えられる。

  • 上総屋不動産：天久保3-4-1
  • ミニストップ：天久保3-3-3
  • しらかし公園：天久保3-7-1
