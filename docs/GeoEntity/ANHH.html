オランダ領アンティル
別名Nederlandse Antillen (@nl)
Category カリブ海地域
Class 行政区画：廃止

概要

オランダ王国の構成国であった。2010年10月10日に解体[1]され、一部オランダの構成国に、一部はオランダ本国の一部となった。

構成地域と現状

地理的には大きく、カリブ海諸島北東部のSSS諸島とカリブ海諸島南西部のABC諸島に分かれていた。アルバ島は1986年に自治領となり[2]、シント・マールテン、キュラソー島も解体時に自治領となった。残りのボネール島、シント・ユースタティウス島、サバ島はオランダ本国の一部であるBES諸島となった。
  • SSS諸島
    • シント・マールテン → 自治領
    • サバ島 → BES諸島
    • シント・ユースタティウス島 → BES諸島
  • ABC諸島
    • ボネール島 → BES諸島
    • キュラソー島 → 自治領
    • アルバ島 → 自治領
