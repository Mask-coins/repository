秋葉原
所在地東京都千代田区
東京都台東区
Category 東京秋葉原エリア
東京都千代田区
Class 地域名

概要

秋葉原駅を中心とする地域。

範囲

概ね北の境界線を蔵前橋通り、東の境界線を昭和通り、西の境界線を昌平橋通り（不忍通り）、南の境界線を神田川とする場合が多い[1][2]。また、千代田区のマスタープラン[3]によると、外神田五丁目・六丁目の地域は、「電気街のにぎやかさのにじみ出しは極力抑え」とあり、秋葉原とは異なる街とする意図が見え，前述の定義を支持する。一方で、他の定義も様々に存在する。東の境界線においては、清洲橋通りを境界線とする考え方もある[4][5]。また、昭和通りから清洲橋通りの間、すなわち千代田区神田佐久間町二丁目～四丁目・神田平河町・神田佐久間河岸・神田和泉町・台東区台東一丁目あたりを裏秋葉原と呼ぶ場合がある。また、神田明神を含んだり、南の境界線を靖国通りとする場合もある[4]。
