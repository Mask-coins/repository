Category 信濃佐久地方
長野県佐久市
Class 町字：大字
藩政村時代は清川村。明治9年に北沢村、山田村とともに常和村を結成するも、明治22年には清川は田口村へ北沢と山田は平賀村に合併となる。その後、合併を経て昭和32年には臼田町大字常和となる（清川地区しかないのに旧3村で結成した常和の名が与えられていた）。平成17年に臼田町が佐久市へ合併される際の地名変更にて、佐久市清川となり、実際上の地名と行政地名が一致することになった。

沿革

  • 田口村（明治22年4月1日～）
  • 田口青沼村（昭和31年9月30日～）
  • 臼田町大字常和（昭和32年4月1日～）
  • 佐久市清川（平成17年4月1日～）
