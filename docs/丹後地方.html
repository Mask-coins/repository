丹後地方
Category 丹後地方
京都府
Class 令制国由来地域名

概要

京都府北部にある地域。旧丹後国を指す場合もあるが、京都府丹後地域のように、旧加佐郡・すなわち舞鶴市と旧大江町のあたりは、丹後地方には含めないこともある。このとき、舞鶴等は中丹地方（中丹地域）に含まれることがある。本サイトでは由良川水系のつながりを重視し、舞鶴等は中丹地方に入れ、丹後地方には含めない立場をとる。

  • 京丹後市
  • 与謝郡与謝野町
  • 与謝郡伊根町
