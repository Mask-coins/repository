奥原宿
Category 東京都渋谷区
東京原宿方面
Class 現代奥○○地名

概要

「奥原宿」は複数の定義がある。

神宮前二丁目商和会（神二商和会）の地域を指す定義がある。

また、ＮＴＴ都市開発株式会社によれば、奥原宿とは表参道と竹下通りの間の大通りに面さない地域を指す[1]。

また、表参道を挟んだ裏原宿の反対側を指す用法もあるらしい。
